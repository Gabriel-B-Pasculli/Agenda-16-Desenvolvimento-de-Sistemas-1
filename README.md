# 🌱 TechVerde – Soluções Sustentáveis

Projeto desenvolvido para a **Agenda 16 – Desenvolvimento de Sistemas I**, utilizando HTML, CSS, JavaScript e o framework W3.CSS.

## 📋 Sobre o Projeto

A **TechVerde Soluções Sustentáveis** é uma empresa fictícia criada para demonstrar a construção de um site institucional moderno, responsivo e interativo.

O site apresenta informações da empresa, parceiros, perguntas frequentes, contato e uma seção de **Serviços / Produtos** com slideshow desenvolvido em JavaScript.

---

## 🚀 Funcionalidades

- Sidebar de navegação fixa
- Página inicial institucional
- Seção de parceiros
- FAQ interativo
- Página de contato
- Slideshow de serviços e produtos
- Navegação manual entre slides
- Navegação automática a cada 3 segundos
- Indicadores de slide
- Pausa automática ao passar o mouse
- Layout responsivo

---

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- W3.CSS
- Font Awesome
